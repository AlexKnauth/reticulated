Reticulated Python

Types and Typechecking for Python

Download as .zip Download as .tar.gz View on GitHub

Reticulated Python is a project that provides static typing in Python where you need it and preserves dynamic flexibility when you don't. It is a linter and source translator written in Python that supports both Python 2 and 3. It finds static type errors where possible, and can also insert runtime checks at the border of typed and untyped code to make sure that your type annotations mean what you think they mean.
